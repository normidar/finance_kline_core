# scripts/fetch_fixtures.py

Binance の公開 API から BTC/USDT の K ライン（ローソク足）データを取得し、
`test/fixtures/` 配下に月別フォルダとして保存するスクリプトです。

## 必要なもの

- Python 3.x（標準ライブラリのみ使用。追加インストール不要）
- インターネット接続（Binance API へのアクセス）

## 実行方法

プロジェクトルートから実行してください。

```bash
python scripts/fetch_fixtures.py
```

## 出力先

```
test/fixtures/
├── 2025_01/
│   ├── kline_5m.csv    # 5分足
│   ├── kline_15m.csv   # 15分足
│   └── kline_1h.csv    # 1時間足
├── 2025_02/
│   └── ...
└── 2025_12/
    └── ...
```

## CSV フォーマット

| カラム       | 内容                        |
|--------------|-----------------------------|
| open_time    | 開始時刻（Unix ミリ秒）     |
| open         | 始値                        |
| high         | 高値                        |
| low          | 安値                        |
| close        | 終値                        |
| volume       | 出来高                      |
| close_time   | 終了時刻（Unix ミリ秒）     |

## カスタマイズ

スクリプト冒頭の定数を変更することで取得対象を調整できます。

| 定数        | デフォルト値              | 説明               |
|-------------|---------------------------|--------------------|
| `SYMBOL`    | `"BTCUSDT"`               | 取引ペア           |
| `INTERVALS` | `["5m", "15m", "1h"]`     | 取得するインターバル |
| `YEAR`      | `2025`                    | 対象年             |
| `MONTHS`    | 1〜12月                   | 対象月と期間       |
